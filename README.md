# 🗓️ Rara Calendar - 나만의 캘린더

음력 표시, 투두 리스트, 그리고 알람 기능이 포함된 개인 일정 관리 웹 앱입니다.

## ✨ 주요 기능

### 📅 캘린더
- **월간 캘린더 뷰**: 한 눈에 전체 일정 확인
- **음력 표시**: 각 날짜의 음력을 함께 표시
- **일정 미리보기**: 날짜 칸에서 일정 제목의 첫 5글자와 추가 일정 개수 표시
- **날짜 선택**: 클릭하여 특정 날짜 선택

### 📝 일정 관리
- **일정 추가/편집/삭제**: 완벽한 CRUD 기능
- **상세 입력**: 제목, 설명, 시간, 카테고리, 알람 설정
- **빠른 네비게이션**: 이전/다음 일정 버튼로 쉬운 이동
- **Today 버튼**: 오늘 날짜로 빠르게 이동

### 🔍 검색
- **일정 검색**: 제목과 설명으로 검색
- **편집 모드**: 체크박스로 여러 일정 일괄 삭제
- **최신순 정렬**: 최신 일정부터 표시

### ✅ 투두 리스트
- **10일 예정 일정**: 오늘부터 10일 뒤까지의 일정 표시
- **슬라이드 패널**: 하단에서 올라오는 애니메이션
- **완료 체크**: 일정별 완료 상태 표시

### ⚙️ 관리자 설정
- **테마 변경**: 파스텔 그레이, 라이트, 다크 모드
- **알람 설정**: 알람 기능 on/off
- **폰트 설정**: 다양한 폰트 선택
- **데이터 초기화**: 모든 데이터 리셋 (주의: 되돌릴 수 없음)

### 🔔 알람 기능
- **시간 기반 알람**: 일정 시간 전 알람 설정
- **브라우저 알림**: Web Notifications API 사용
- **PWA 지원**: 설치하여 앱처럼 사용 가능

## 🚀 시작하기

### 설치 및 실행

```bash
cd frontend
npm install
npm run dev
```

개발 서버는 `http://localhost:5173/` 에서 실행됩니다.

### 빌드

```bash
npm run build
```

프로덕션 최적화 버전으로 빌드됩니다.

### 미리보기

```bash
npm run preview
```

빌드된 버전을 로컬에서 미리보기합니다.

## 💾 데이터 저장

- **localStorage 기반**: 모든 데이터는 브라우저의 localStorage에 저장됩니다
- **자동 저장**: 30초마다 자동으로 저장됩니다
- **오프라인 지원**: PWA 방식으로 오프라인에서도 접속 가능합니다

## 🛠 기술 스택

### 프론트엔드
- **React 18**: UI 프레임워크
- **TypeScript**: 타입 안전성
- **Vite**: 빠른 빌드 도구
- **Tailwind CSS**: 유틸리티 기반 스타일링
- **Day.js**: 날짜/시간 관리
- **Zustand**: 전역 상태 관리
- **lunar-calendar**: 음력 변환

### 기능
- **Web Notifications API**: 알람 기능
- **Service Workers**: PWA 지원
- **localStorage**: 클라이언트 데이터 저장

frontend/

│   ├── components/          # React 컴포넌트

│   │   ├── EventEditor/     # 일정 추가/편집
│   │   ├── EventDisplay/    # 일정 표시 및 제어

│   │   ├── TodoList/        # 투두 리스트
│   │   ├── AdminPanel/      # 관리자 설정
│   │   └── Header/          # 헤더 네비게이션
│   ├── hooks/               # 커스텀 React 훅
│   │   ├── useLocalStorage.ts
│   │   └── useNotification.ts
│   ├── utils/               # 유틸리티 함수
│   │   └── dateHelper.ts

│   │   └── eventStore.ts
│   ├── types/               # TypeScript 타입
│   │   └── event.ts
│   ├── styles/              # 스타일시트
│   │   └── responsive.css
│   ├── App.tsx              # 메인 앱 컴포넌트
│   ├── index.css            # 전역 스타일
│   └── main.tsx             # 진입점
├── public/
│   ├── manifest.json        # PWA 메니페스트

## 🔒 개인정보 보호

- 모든 데이터는 브라우저의 localStorage에만 저장됩니다
- 서버로 전송되는 데이터가 없습니다
- 100% 오프라인 지원 가능합니다

## 📱 반응형 디자인

- **모바일 (< 640px)**: 최적화된 터치 인터페이스
- **태블릿 (640px ~ 1024px)**: 중간 크기 화면 대응
- **데스크톱 (> 1024px)**: 풀 기능 인터페이스

## 🌐 PWA 지원

- **설치 가능**: 홈 화면에 앱 아이콘 추가
- **오프라인 작동**: Service Worker로 오프라인 접속 지원
- **빠른 로딩**: 캐시된 리소스로 빠른 로딩 속도
